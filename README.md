## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/FedericoGabrielCastro/Brastlewark.git
```

2. Change the working directory

```bash
cd Brastlewark
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```

🌟 You are all set!
